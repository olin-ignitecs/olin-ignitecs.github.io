So now what? We'd like to help you continue to learn and create with code.

Arduino

Arduinos are little computers that you can program with c/c++ language. If you're interested in buying your own Arduino, Amazon sells some for $15.

JavaScript

Javascript is the language that makes the web interactive. It can be used to create animations!

Arduino Casino Games

In the workshop you did, you controlled lights with buttons to create patterns and a game of roulette. You can find the resources from that workshop through these links: Introduction and Toolbox.

PaperJS Animations

PaperJS is a library of Javascript (subsection of the language that's been simplified) that was specifically created for making animations. In the workshop you worked on these challenges with the help of this toolbox. If you're interested in doing more with PaperJS, you can work of the template.
